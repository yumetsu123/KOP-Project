# Privacy Policy

## 1. Introduction

This Privacy Policy explains how the Simon Game application handles user data.

This project was created for educational purposes as part of a university course assignment.

---

## 2. Data Collection

The Application does **not** collect, store, process, or transmit any personal data to external servers.

No user accounts are created.
No email addresses are collected.
No analytics tools are used.
No tracking services are integrated.

---

## 3. Local Storage Usage

The Application uses the browser's **localStorage** feature to store:

- Game progress
- Player score
- User identifier (entered locally)
- Cookie consent status

All stored data remains on the user's device and is never transmitted to any third party.

Users may clear this data at any time by clearing their browser storage.

---

## 4. Cookies

The Application uses a single technical cookie to store the user's cookie consent preference.

This cookie:
- Does not track users
- Does not contain personal information
- Is used only to remember the user's consent decision

---

## 5. GDPR Compliance

In accordance with the General Data Protection Regulation (GDPR):

- The Application does not collect personal data.
- Users are informed about the use of cookies.
- Users have the right to clear local storage and cookies at any time.

Since no personal data is processed, no data controller or data processor activities are performed.

---

## 6. Third-Party Services

The Application does not use any third-party tracking, analytics, or advertising services.

All dependencies used in the project are open-source libraries required for application functionality.

---

## 7. Changes to This Policy

This Privacy Policy may be updated if the functionality of the Application changes.

---

## 8. User Responsibilities

By using this Application, users agree to:

- Use the Application for lawful and educational purposes only.
- Not attempt to reverse engineer, modify, or exploit the Application in any malicious way.
- Not use the Application in a way that could damage, disable, or impair its functionality.
- Understand that all game data is stored locally and may be permanently deleted if browser storage is cleared.

Users are responsible for managing their own browser data, including cookies and localStorage.
