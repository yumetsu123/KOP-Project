# Simon Game

Web-based implementation of the classic **Simon memory game** built with React and Vite.

The goal of the game is to repeat an increasing sequence of colors.  
Each round adds a new color to the sequence.  
If the player selects the wrong color â€” the game ends.

## Technologies Used

- React
- Vite
- Redux Toolkit
- JavaScript (ES6+)
- CSS
- Storybook (UI component documentation)
- JSDoc (generated code documentation)

## Installation & Setup

### Clone repository

```bash
git clone <https://github.com/yumetsu123/Lab5>
cd Lab5-main
```

### Install dependencies:

```bash
npm install
```

### Run locally

Start the development server:

```bash
npm run dev
```

The application will be available at http://localhost:5173.

## Storybook

Run Storybook for component documentation:

```bash
npm run storybook
```
Storybook is used to document and demonstrate reusable UI components.

## Documentation

- Generated Documentation (JSDoc)
Run:
```bash
npm run docs
```
